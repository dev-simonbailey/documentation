Vehicle Details

The Vehicle details page is divided into 7 areas

Map

Latest incidents

Fleet history

Site history

Hire status

Map

The map shows all the latest incidents for the vehicle selected. Zooming in will allow for a more detailed view.

All the normal map functions are available.

Overview

The overview section gives details on the following:

  • Icon showing the machine type
  • Vehicle registration
  • Last online time
  • Hire status
  • Last know location (Lat & Long) - with link to show the last known position
  • Number of RED incidents today, for the selected vehicle
  • Number of YELLOW incidents today, for the selected vehicle
  • The current fleet the vehicle is assigned to
  • The current site the vehicle is assigned to

The CAMERA icon gives access to the live feed, from the vehicle's cameras

When clicked, it will then display the live feed modal, and clicking on any of the available cameras will initiate the live feed stream from the vehicle.

Latest Incidents

This table details the latest incidents, for the vehicle and has actions that can be performed.

The details are:

  • Time of incident
  • Severity of incident
  • Site incident occurred at
  • Location - Lat & Long
  • Actions
    • View the incident details
    • Request incident video download (available if feature eanbled in settings)
    • Play button to play downloaded incident videos (Where a requested video is available)
    • View incident image

Actions

View the incident details

  • Please refer to Incident Details for further information

Request incident video download

Clicking on the CAMERA icon will display a context menu where a view or download request can be made that requires a selection as to the purpose of the video download request

  • Once a selection has been made, the context menu shows a selection that is required as to the purpose of the video download/view request

  • Making a selection will send a video download request to the administrator

Play downloaded video

  • Clicking on the PLAY icon will open a new tab and play the downloaded video.

View incident image

  • Clicking on the IMAGE icon will display the image attached to the incident.

Fleet History

The fleet history table shows the movement of the vehicle across fleets

Pagination is available at the bottom of the table

Site History

The site history table shows the movement of the vehicle across sites

Pagination is available at the bottom of the table

Hire Status

The hire status table shows the hire status of the vehicle

Pagination is available at the bottom of the table

The Hire Status table gives access to additional features

  • Add a hire
  • Edit a hire
  • Delete a hire

Add a hire

This feature is detailed in the Hiring section

Edit a hire

This feature is detailed in the Hiring section

Delete a hire

If the TRASH icon is clicked the selected hire will be deleted

Deleting a hire is effective immediately, there is no confirmation required, so care is to be taken.
