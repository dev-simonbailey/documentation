Reports

The report generator allows tenants to generate reports that are emailed to a specified email address at a scheduled time, containing data that is defined in the Report Wizard, accessed via the “Reports” menu item.

The report system utilises a Task Sceduler, within the app, that collects the reports that are to be run that day. This tasks runs at 05:00 every day.

All the reports that are identified, as needing to be run that day, are held in a dispatch queue that monitors the time and when a report dispatch time matches the current time, the report is created and dispatched to the recipient.

A point of note is that if a report is created today, at 09:00, and the user specifies a dispatch time of 15:00, the report will not get queued until 05:00 the following day, and, therefore, will not run until 15:00 the following day.

Creating a report

  • Navigate to the Reports page, using the Reports item in the left hand navigation menu.

  • Click on the “Create Report” button.

  • A modal appears that displays the Report Wizard.

  • Select a Timeframe, from the drop-down - Over what time period do you wish to have data for.

  • Select a re-occurrence - How often the report is to be generated.

  • Select a dispatch Time - What time the report is to be generated.

  • Give the report a name - It's worth naming reports in a meaningful way.

  • Specify the subject line of the email that will be sent, with the report attached.

  • Add the recipients of the report - To add multiple, enter the email address and press the enter key, then add another email address, and so on.

Press the NEXT button to move to the next step

  • In this step, you can specify the Tenants that should be included in your report.

Press the NEXT button to move to the next step

  • In this step, you can specify the Sites that should be included in your report.

Press the NEXT button to move to the next step

  • In this step, you can specify the Vehicles that should be included in your report.

Press the CREATE button to complete the wizard and add your report

  • You will now see your report in the Reports Page.

You can edit your report, at any time, and the changes will only be applied the next time your report is scheduled to be run.

If you delete a report, it cannot be recovered, so only delete reports that you do not want.
