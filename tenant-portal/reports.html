Reports

The system provides two ways to generate reports from the Dashboard:

  • Generate Report - Creates an instant PDF report that downloads directly to your device
  • Schedule Report - Sets up automated reports that are emailed to specified recipients on a schedule

Generate Report (Instant Download)

The Generate Report button creates an instant PDF report based on the current Dashboard view and downloads it directly to your device.

How to Generate an Instant Report

1. Navigate to the Dashboard

2. Apply the desired filters (Site, Fleet, Date, Machine Type)

3. Click the Generate Report button

4. The PDF will be generated and automatically downloaded to your device

Report Contents

The generated PDF includes:

  • Dashboard widgets (Red Alerts, Yellow Alerts, Critical Vehicle/Fleet/Site)
  • Applied filters summary
  • Latest Incidents chart and bubble diagram
  • Top 5 vehicles with the most alerts
  • Total alerts per fleet
  • Total alerts per site

The report filename includes the current date and time, making it easy to identify when the report was generated.

Schedule Report (Automated Email)

The Schedule Report button allows you to create automated reports that will be generated and emailed to specified recipients on a regular schedule.

Note: The Schedule Report button is only available when a preset date filter is selected (Today, Last 7 Days, Last 30 Days, etc.). It is disabled when using a custom date range.

Important: The output of scheduled reports will not take account of any specific user permissions for recipients. Please carefully consider the people receiving this report.

Creating a Scheduled Report

To create a scheduled report, click the Schedule Report button on the Dashboard. A dialog window will appear with the following options:

Report Name

Give the report a meaningful name that describes its purpose or content. This helps identify reports in the list.

Recipients Email Address

Enter the email addresses where the report will be sent. Multiple email addresses can be entered, separated by a semicolon (;).

Example: user1@example.com;user2@example.com

Email Subject

Enter the subject line for the email. This is optional but recommended for easy identification of report emails.

Report Occurrence

Three scheduling options are available:

Daily

This option allows you to select specific days of the week when the report should be generated, along with the time of day.

For example, you can set the report to run every weekday at 8:00 AM.

Weekly

This option generates the report once per week on a specific day and time.

Select the day of the week and the time when the report should be generated and sent.

Monthly

This option generates the report once per month on a specific day and time.

Select the day of the month (1-31) and the time when the report should be generated and sent.

Report times are available in 15-minute intervals from 00:15 to 23:45.

Include Attachments

Include CSV file

Enable this option to include a CSV file with the incident data alongside the PDF report. This allows recipients to analyse the data in spreadsheet applications.

Saving the Report

Once all options are configured, click the Save button to create the scheduled report.

The report will be created with the current Dashboard filters (Site, Fleet, Machine Type) applied automatically.

Report Email

When a scheduled report is sent, recipients will receive an email containing:

  • The PDF report as an attachment
  • Optionally, a CSV file with incident data (if enabled)

Managing Scheduled Reports

Scheduled reports can be viewed and managed from the Reports section in the main navigation menu.

Reports Table Overview

The table displays all configured scheduled reports with the following information:

  • ID - Unique identifier for the report
  • Report name - The name assigned to the report
  • Created By - The user who created the report
  • Recipients - Email addresses receiving the report
  • Reoccurrence - Schedule frequency (Daily, Weekly, or Monthly)
  • Last sent at - Date and time of the last successful delivery
  • Actions - Edit and Delete buttons

Editing a Report

Click the Edit icon (pencil) to modify an existing scheduled report.

When editing a report, you can modify the following settings:

  • Report name
  • Recipients email addresses
  • Email subject
  • Report occurrence (Daily, Weekly, Monthly)
  • Schedule settings (days and time)
  • Include CSV file option

Note: The data filters (Site, Fleet, Machine Type) that were configured when the report was created cannot be changed. To use different filters, you need to create a new report from the Dashboard with the desired filter settings.

Deleting a Report

Click the Delete icon (trash) to remove a scheduled report.

Deleting a report is permanent. The report will no longer be sent to recipients.
