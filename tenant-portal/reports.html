Reports

The Reports page displays all scheduled reports that will be automatically generated and emailed to specified recipients based on their schedule. Each email contains a PDF file by default, and optionally, a CSV data file.

Reports Table

The reports table displays all scheduled reports along with their configuration and available actions.

The table includes the following columns:

  • ID
    • Unique identifier of the report.
  • Report Name
    • The name given to the report when it was created.
  • Created By
    • The user who created the report schedule.
  • Recipients
    • Email addresses the report is sent to. Multiple recipients are separated by a semicolon.
  • Reoccurrence
    • The selected schedule for the report - Daily, Weekly, or Monthly.
  • Last Sent At
    • The date and time when the report was last sent.
  • Actions
    • Edit - Modify the schedule or recipients of the report.
    • Delete - Remove the report schedule.

Report Occurance

Three options are available

Once per day

This option allows the days the report is to by run, along with the time the report is to be generated

Once per week

This option allows the day of the week the report is to be run, along with the time the report is to be generated

Once per month

This option allows the day of the month the report is to be run, along with the time the report is to be generated

Save or Cancel Changes

After editing the report, click Save to apply your changes or Cancel to discard them.

Note: Editing a report does not change the data scope. To modify the data included in the report, create a new report from the Dashboard page.

Note: The reports generated will not respect user-specific permissions. Ensure recipients are carefully selected.

Delete Report

Clicking the Delete icon will permanently remove the scheduled report.
