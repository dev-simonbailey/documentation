Incidents

The Incidents page is divided into three areas:

  • The map
  • Filtering and global actions
  • Incidents list

Map

The map shows the last known GPS location of each vehicle. Zooming in will show a more precise location.

When the page loads, initially, it will default to no filtering and todays incidents.

The normal map controls are available.

Filtering and Global Actions

The data displayed on the incidents page can be filtered with the following filters:

Search by registration

By Date

Three preset date ranges are available, but any date range can be set, by navigating in the date selector and selecting the date range manually

Alert Type

Filter by Type has 3 options

  • Red
    • Show only RED alerts
  • Amber
    • Show only YELLOW alerts
  • Authorised entry
    • Show alerts where the operator authorised entry into the detection zones, irrespective of the alert type generated.

Red and Yellow alerts can be shown together, by selecting both checkboxes (this is the default behaviour fo the page when it initially loads).

If Authorised entry is selected, then the RED and YELLOW alerts options are deselected and only results where entry into the detection zones was authorised are shown

By Fleet

By Site

Multiple fleets may be selected

Incidents List

The incidents table details the following:

  • Time - The time the incident occurred
  • Alert Type - Red / Yellow / Authorised entry
  • Registation - The registration of the vehicle
  • Fleet - The fleet the vehicle is currently assigned to
  • Site - The site the vehcile is currently assigned to
  • Actions - View Incident, Request Incident video
  • Device status - Green (Online), Red (Offline)
  • Image loading status

Actions

View Incident Details

Clicking on the View icon will display the vehicle details page

Further details can be found on the Incident Details page

Request Incident Video

Clicking on the Camera icon will display a context menu where a view or download request can be made that requires a selection as to the purpose of the video download request

Once a selection has been made, the context menu shows a selection that is required as to the purpose of the video download/view request

Making a selection will send a video download request to the administrator

Image status

Image requested

Image loaded. When clicked, it will display the image preview modal.

Pagination

Several options are available to move through the list of devices

  • Selecting the number of devices to show, via the drop-down

  • Using the pagination navigation controls
