Fleets

The Fleets page details the fleets that have been created in the portal, and show the following:

  • The Fleet Name
  • If the Fleet is enabled
  • Actions that can be performed on existing fleets
  • Add a new fleet
  • Import a fleet from a CSV
  • Export the fleet(s) to CSV

Actions

EDIT FLEET

Clicking on the Pencil icon will show the Edit Fleet modal

On this modal, the name (Title), the company this fleet belongs to, the description, the enabled status and fleet manager can be edited. Clicking on SAVE will save any modifications, whilst clicking on CANCEL will stop the editing process and return to the Fleets list.

REMOVE FLEET

Clicking on the Trash will display a confirmation that the selected fleet is to be deleted.

ADD FLEET

Clicking in the ADD FLEET button will display the Create Fleet modal

To create a fleet, follow these steps

  • Step 1 - Enter a name for the fleet
  • Step 2 - Select a company the fleet will belong to
  • Step 3 - Enter a description of the fleet
  • Step 4 - Enable the fleet within the portal
  • Step 5 - Select a fleet manager
  • Step 6 - Save the new fleet

Clicking on SAVE will save the new fleet, whilst clicking on CANCEL will stop the creation process and return to the Fleets list.

Import From CSV

Click on the IMPORT FROM CSV button to open the Import Fleets modal

Click on the paperclip or within the file box to show the file chooser and navigate to the location on your computer where the CSV file is located.

Clicking on the SAVE button will upload the file and process the data, adding it to the portal. Clicking on the CANCEL button will cancel the import process and return to the Fleet list.

Fields

  • id
    • Integer
    • Example: 1
  • title
    • String
    • Example: Safety Sheild
  • description
    • String
    • Example: Ratione vel itaque maxime et. Ad omnis...
  • enabled
    • Integer
    • Example: 1 (enabled), 0 (not enabled)

Export Fleet To CSV

To export the list of fleets, click on the EXPORT CSV button. The CSV export process will then start and download the created CSV file to your computer, in to the location specified in your browser.

The normal location for downloading, in the Chrome browser, is your Downloads folder.
