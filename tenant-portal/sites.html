Sites

The sites page gives details of the sites already created, as well as offering the ability to add new sites, either singly or via CSV import, as well as exporting the sites data to CSV.

Searching by Project Code

You can search for a specific site using the Project Code (Main Project Number)

Site Details

Each site is detailed in the rows of the table and shows information as well as actions that can be taken.

Columns

  • Title
    • The Site Name
  • Address
    • The Site Address
  • Enabled
    • The status of the site in the portal
  • Actions
    • View Site Dashboard
    • Edit Site
    • Remove Site

Actions

View Site Dashboard

If clicked, a new tab will open and the sites dashboard will be shown.

Edit Site

If clicked, the Edit Site modal is displayed, where details of the site can be edited.

Remove Site

If clicked, a modal is shown that requires confirmation of the removal

Adding a Site

Clicking on the ADD SITE button will open the Create Site modal.

To add a site, follow these steps.

Step 1 - Enter a name for the site

Step 2 - Setting Boundaries

The site address is required, and can be searched for by postcode, as well as full address.

There are two options for setting the boundaries

1. Selecting a radius.

Moving the slider to the right will increase the radius of the boundary.

2. By defining a boundary.

Pressing CONFIRM will set the custom boundary, clicking on Pressing CONFIRM will set the custom boundary, clicking on CLEAR will clear the custom boundary defined, allowing a new custom boundary to be created will clear the custom boundary defined, allowing a new custom boundary to be created.

To create a custom boundary, click on the map, adding corner points to define the custom boundary. Markers can be moved, once placed, by clicking and dragging them to the desired location.

Step 3 - Selecting the company the site belongs to

Step 4 - Adding additional information

Step 5 - Entering the project name and number

Step 6 - Enabling the site within the portal

Step 7 - Selecting a site manager

Step 8 - Saving the new site

Clicking on SAVE will save the new site to the portal, clicking CANCEL will stop the add new site process and return to the sites list.

Importing Sites from a CSV

Clicking on the IMPORT FROM CSV button will open the Import Sites modal.

Clicking on the paperclip, or within the file box, a CSV file can be selected form the local computer

Clicking on SAVE will import the sites to the portal. Clicking on CANCEL will stop the import process and return to the sites list.

The CSV file needs to be formatted in the following manner:

id,title,address,latitude,longitude,radius,notes,main_project_name,main_project_number,enabled

Fields

  • id
    • integer
    • Example: 1
  • title
    • string
    • Example: Safety Shield Office
  • address
    • string
    • Example: Road Four, Winsford Industrial Estate, Winsford CW7 3RS, UK
  • latitude
    • decimal
    • Example: 53.2721833
  • longitude
    • decimal
    • Example: -2.82148443
  • radius
    • decimal
    • Example: 0.5
  • notes
    • string
    • Example: A velit et tenetur voluptatum...
  • main_project_name
    • string
    • Example: My Project
  • main_project_number
    • string
    • Example: 1456
  • enabled
    • integer
    • Example: 1 (is enabled), 0 (is not enabled)

Exporting Sites to a CSV

To export the list of sites, click on the EXPORT TO CSV button. The CSV export process will then start and download the created CSV file to your computer, in to the location specified in your browser.

The normal location for downloading, in the Chrome browser, is your Downloads folder.

Change Hierarchy

The hierarchy of the sites can be changed, to create associations with, click on the CHANGE HIERARCHY button, to display the hierarchy modal

By default, a site will be at the parent level of the hierarchy, but if desired, a site can be associated to another site (sub-site).

To save the modified hierarchy, click on the SAVE button, clicking on the CANCEL button will stop the change hierarchy process and return to the sites list.
