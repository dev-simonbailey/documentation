Hiring

The Hiring page details the vehicles that are hired from the tenant along with those vehicles hired to the tenant.

The page is split into two distinct areas, via tabs.

My vehicles on hire

The My Vehicles On Hire table shows, initially, the vehicles owned by the tenant that are on hire to other tenants.

To include historical hires, i.e. where the hire has ended or expired, check the show historical checkbox.

  • ID of the hire
    • This is an internal reference in the portal
  • Registration
    • The registration of the vehicle on hire
  • Hired to
    • The tenant that has hired the vehicle
  • Start time
    • The date / time the hire started
  • End time
    • The date / time the hire ended. If no date is shown, then the hiring period has an unspecified end date.
  • Is active
    • The hire is active status
  • Actions
    • Edit - view the hire details
    • Delete Hire - end the hire

Hired to ACTIONS

Edit Hire>

Clicking the PENCIL icon will open the Edit Vehicle On Hire modal

The modal shows the vehicle registration, along with when the hire period started and ended (no end date is shown if the Unspecified end date toggle is enabled)

All of the hiring options can be edited, but note the end date and time must be in the future, relative to the start date and time.

Start Immediately

If the Start Immediately toggle is enabled, the hiring start date and time will be now, and the Start Date date picker will be disabled.

Unspecified End Date

If the Unspecified End Date toggle is enabled, the hiring end date and time be left empty, and the End Date date picker will be disabled.

Selecting a date and time

Using the date picker, you can select a date

and to select a time, click on the Clock icon

and the date picker will swap to the time picker

and for speed, clicking on the time values will open a quick selection menu

To return to the date picker, click on the Calendar icon

The same operations are available for the End date

Clicking the SUBMIT button which will save the updates

Add Hire

To add a hire, click on the ADD HIRE button, which will display the Add hire modal

To add a vehicle(s) to a hire, follow these steps

Step 1 - Selecting vehicle(s)

Multiple vehicles may be selected

Step 2 - Selecting tenant to hire to

Only tenants that have been setup as Hire to, can be selected. If a tenant that is required is not present in the Select tenant list, then please contact your administrator

Step 3 - Selecting Start and End dates and times for the hire

If the hire period is to start immediately, then enabling the Start Immediately toggle will set the start date and time to now.

If the hire period will not have a defined end date and time, the Unspecified End Date toggle should be enabled.

If the hire start date and time are to be selected, use the date/time picker.

and to select a time, click on the Clock icon

and the date picker will swap to the time picker

and for speed, clicking on the time values will open a quick selection menu

To return to the date picker, click on the Calendar icon

Repeat for the End Date, if applicable

Step 4 - Enabling the hire

Clicking the SUBMIT button will save the updates

Deleting a hire

To delete a hire, click on the TRASH icon

The delete operation will delete the hire from the hiring tenant and the hired to tenant. There is no confirmation, so care should be taken

Errors and Warnings

As a hire period for a vehicle cannot overlap another hire period for the same vehicle, the modal will display notices if it detects any errors.
