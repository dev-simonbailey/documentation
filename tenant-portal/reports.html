Reports

The report generator allows reports to be generated that are emailed to a specified email address at a scheduled time.

Creating a report

Report Name

Give the report a name - It's worth naming reports in a meaningful way

Recipients Email Address

Enter the email address the report will be sent

Subject Line

The subject line for the email

Report Occurance

Three options are available

Once per day

This option allows the days the report is to by run, along with the time the report is to be generated

Once per week

This option allows the day of the week the report is to be run, along with the time the report is to be generated

Once per month

This option allows the day of the month the report is to be run, along with the time the report is to be generated

Report Period

Four options are available

  • This report will include all incidents from the previous day
  • This report will include all incidents from the previous 24 hours
  • This report will include all incidents since the report was last run
  • This report will include all incidents from the day the report is run

Create report

Click on the CREATE button to create the report
