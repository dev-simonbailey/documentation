Vehicle Details

The Vehicle details page is divided into 3 areas

Map

Hire status

Map

The map shows the latest known location for the vehicle.

All the normal map functions are available.

Overview

The overview section gives details on the following:

  • Icon showing the machine type
  • Vehicle registration
  • Last online time
  • Hire status
  • Last know address
  • Number of RED incidents today, for the selected vehicle
  • Number of YELLOW incidents today, for the selected vehicle
  • Number of AUTHORISED incidents today, for the selected vehicle
  • Devices currently assigned to the vehicle
  • The current fleet the vehicle is assigned to
  • The current site the vehicle is assigned to
  • View Incidents button â€“ when clicked, incidents page for the vehicle in the last 7 days will be displayed.

The CAMERA icon gives access to the live feed, from the vehicle's cameras

When clicked, it will then display the live feed modal, and clicking on any of the available cameras will initiate the live feed stream from the vehicle.

Hire Status

The hire status table shows the hire status of the vehicle

Pagination is available at the bottom of the table

The Hire Status table gives access to additional features

  • Add a hire
  • Edit a hire
  • Delete a hire

Add a hire

This feature is detailed in the Hiring section

Edit a hire

This feature is detailed in the Hiring section

Delete a hire

If the TRASH icon is clicked the selected hire will be deleted

Deleting a hire is effective immediately, there is no confirmation required, so care is to be taken.
