Dashboard

The Dashboard is split into seven areas, detailing general and specific data as well as filtering.

Map

The map shows the geographical position that was recorded when an incident took place.

At various zoom levels, the incidents are grouped, visually, zooming in will show more detail.

When incidents have occurred in close proximity to each other, the incidents are grouped and displayed as a grouping icon.

Zooming in further show the incidents, in isolation.

Clicking on an incident marker, will show an overview of the incident.

The usual Maps controls are available.

Filtering

The data shown on the dashboard can be filtered using the available filters.

By Fleet

Number of incidents

The upper section displays the incident count in numeric format, based on the selected filters

Latest Incidents

The upper section displays the latest incidents, based on filtering, in graphical chart format

Vehicles

This table details the vehicles and the incidents they have been involved with, based on the filtering.

There is a the functionality to show all vehicles by clicking on Show All Vehicles at the bottom of the section.

Fleets

This table details the incidents by Fleet.

There is a the functionality to show all fleets by clicking on Show All Fleets at the bottom of the section.

Sites

This table details the incidents by Site.

There is a the functionality to show all sites by clicking on Show All Sites at the bottom of the section.

Actions

Scheduling a Report

Clicking on the SCHEDULE REPORT button will open the Report Scheduling form.

To schedule a report, follow these steps

Step 1 - Enter the Report Name REQUIRED

Step 2 - Enter the Recipients' Emails REQUIRED

Multiple email addresses can be separated by a semicolon.

Step 3 - Add an Email Subject OPTIONAL

Step 4 - Select the Report Occurrence (Daily, Weekly, Monthly) REQUIRED

Step 5 - Choose the Days and Time to Receive the Report REQUIRED

Step 6 - Enable CSV Attachment if required OPTIONAL

Selecting this option will include the report data as an additional CSV file attachment in the email.

Step 7 - Save the scheduled report REQUIRED

Note: The reports generated will not respect user-specific permissions. Ensure recipients are carefully selected.

Important: The filters applied on the Dashboard page (Date Range, Fleets, Sites, and Machine Types) define the data included in the automated report.

Make sure to set the desired filters before scheduling the report to ensure the correct data is captured.

Note: The scope of a scheduled report (which data is included) can only be changed by creating a new report from the Dashboard page. Editing a scheduled report will not change the data scope.

Please refer to the Reports section for editing or deleting scheduled reports.

Generating a Report

Clicking on the GENERATE REPORT button will immediately generate the report based on the current filters.

The report will be generated and automatically downloaded as a PDF file.
