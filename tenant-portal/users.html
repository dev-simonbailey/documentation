Users

The User tab in the Settings page details the current users of the portal, along with the ability to add new users and edit existing users

Operations that can be performed

  • Create User
  • Invite User
  • Import users from CSV
  • Export users to CSV
  • Edit existing user
  • Login as a particular user
  • Remove a user

Creating a user

Clicking the CREATE USER button will display the Create User page

Step 1 - Enter the user's name

Step 2 - Enter the user's email

Step 3 - Enter the user's password

Step 4 - Confirm the password by entering it into the confirmation box

If the passwords do not match then a message will display. Please ensure that the passwords match

Step 5 - Register the user

Click on REGISTER to register the user in the portal, click on CANCEL to dis-regard the new user and return to the users list

Inviting a user

Clicking the INVITE USER button will display the Invite User modal

Step 1 - Enter the user's email

Step 2 - Select the team the user will be in, from the drop-down list

Step 3 - Send Invite

Click on SEND INVITE to register the user in the portal, click on CANCEL to dis-regard the invite and return to the users list

Importing users from CSV

Clicking the IMPORT USERS FROM CSV button will display the Import Users from CSV modal

Step 1 - Select the CSV file to import

Step 2 - Select the CSV file containing the users to be imported, by clicking on the Paperclip icon on in the file box.

Step 3 - Select the CSV file from your computer

Step 4 - Save the users to the portal

Click on SAVE to import the user CSV file and process the users, click on CANCEL to dis-regard the import and return to the users list

Fields

  • id OPTIONAL
    • This value is automatically appiled by the database
  • name REQUIRED
    • string
    • Example: John Smith
  • email REQUIRED
    • string
    • Example: john.smith@company.com
  • password REQUIRED
    • string
    • Example: PY(FArt3pen<(Ebs
  • enabled REQUIRED
    • integer
    • Example: 1
    • NOTE: If set to 1 then the device will be enabled in the portal, it set to 0 then the device will not be enabled in the portal

Exporting users to CSV

Clicking the EXPORT USERS TO CSV button will download a CSV file containing the users in the portal

The normal location for downloading, in the Chrome browser, is your Downloads folder.

Actions

Edit

Clicking the PENCIL button will display the Edit User modal

On this modal, the following can be edited

  • Name - The name of the user
  • Email - The email address of the user
  • Enabled - The enabled status of the user in the portal
  • Data Protection Officer - Assign the user as a DPO, to allow the authorisation of incident data to be downloaded.

Clicking the SAVE button will update the user, whilst clicking the CANCEL button will dis-regard any changes and return the user list

Log-in as user

Clicking the LOG-IN button will log the user into the portal as the selected user

Remove user

Clicking the TRASH CAN button will remove the user from the portal

Deleting a user will immediately delete them from the portal - there is no confirmation dialog shown, so care should be taken

NOTE: If a users is deleted by mistake, please contact your administrator and ask if the user can be re-instated, as the delete mechanism is a soft delete.
