Incident Details

The Incident details page is divided into 4 areas

Map and Overview

Recent Incidents

Recent Incidents at this site

Map

The map shows all the latest incidents. Zooming in will allow for a more detailed view.

All the normal map functions are available.

Overview

The overview section gives details on the following:

  • Icon showing the machine type
  • Registration of the vehicle
  • Date and time of the incident
  • Incident Severity - Red / Yellow
  • The site incident occurred at
  • An Incident Video Download Request button

Incident Video Download Request

  • To request a download a video of the incident, click on the DOWNLOAD REQUEST button

  • Select the request type

  • and the reason for the request

  • Making a selection will send a video download request to the administrator

Recent Incidents

This table details the most recent incidents

The details are

  • Time of incident
  • Severity of incident
  • Site incident occurred at
  • Location - Lat & Long
  • Actions
    • View the incident details
    • Request incident video download
    • Play button to play downloaded incident videos, if applicable
    • View Incident image, if applicable

Actions

View the incident details

Clicking on the View icon will load the selected incident into the current page

Request Incident Video Download

Clicking on the Camera icon will display a context menu where a view or download request can be made that requires a selection as to the purpose of the video download request

If a Tickis present, instead of the Camera icon, an incident video download request has been made.

Once a selection has been made, the context menu shows a selection that is required as to the purpose of the video download/view request

Making a selection will send a video download request to the administrator

Play Downloaded Video

Clicking on the Play icon will play the downloaded video

View Incident Image

Clicking on the Image icon will display the incident image

If an ORANGE indicator is present, then the image has been requested
