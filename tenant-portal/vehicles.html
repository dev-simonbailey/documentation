Vehicle

The Vehicles page is divided into three areas:

  • The map
  • Filtering and global actions
  • Vehicle list

Map

The map shows the last known GPS location of each vehicle. Zooming in will show a more precise location.

The normal map controls are available.

Filtering and Global Actions

The data displayed on the vehicles page can be filtered with the following filters:

Search by registration

Select Site

Select Fleet

Select Machine Type

Hiring Status

Actions

Adding a Vehicle

Clicking on the ADD VEHICLE button will show the Create Vehicle modal

To add a vehicle, follow these steps

Step 1 - Enter a Registration/Fleet Number REQUIRED

Step 2 - Enter a VIN Number OPTIONAL UNIQUE

Step 3 - Enter a PIN Number OPTIONAL

Step 4 - Select a company that the vehicle belongs to OPTIONAL

Step 5 - Select a fleet that the vehicle belongs to REQUIRED

Step 6 - Select a site that the vehicle belongs to OPTIONAL

Step 7 - Select a vehicle type OPTIONAL

Step 8 - Select a Make and Model OPTIONAL

Step 9 - Select the device, from the UUID, that is installed in the vehicle REQUIRED

Step 10 - Enable the vehicle in the portal REQUIRED

Step 11 - Auto assign the vehicle back to the depot OPTIONAL

Step 12 - Save the new vehicle REQUIRED

Importing vehicles from a CSV file

Click on IMPORT FROM CSV to display the Import Vehicles modal

Clicking on the paperclip, or within the file box, a CSV file can be selected from the local computer

Fields

  • id OPTIONAL
    • Integer
    • Example: 1
  • uuids REQUIRED
    • string
    • Example: G4CE0400437
    • Multiple UUIDS can be assigned to a vehicle by seperating the uuids with a space
    • Example: G4CE0400437 G4CE0900396
  • fleet_id OPTIONAL
    • integer
    • Example: 1
  • site_id OPTIONAL
    • integer
    • Example: 1
  • registration OPTIONAL
    • string
    • Example: 99C 0F5
  • machine_type_id OPTIONAL
    • string
    • Example: Excavator
  • enabled OPTIONAL
    • integer
    • Example: 1 (enabled), 0 (not enabled)
  • auto_assign_to_depot OPTIONAL
    • integer
    • Example: 1
  • hired_from OPTIONAL
    • string
    • Example: STAGING DEMO COMPANY
  • hired_to OPTIONAL
    • string
    • Example: STAGING TEST 2
  • VIN OPTIONALUNIQUE
    • string
    • Example: HDG39283923IDJ232
  • PIN OPTIONAL
    • string
    • Example: 1234
  • last_known_lat OPTIONAL
    • decimal
    • Example: 53.19683000
  • last_known_lon OPTIONAL
    • decimal
    • Example: -2.89304000

Export to CSV

To export the list of vehicles, click on the EXPORT TO CSV button. The CSV export process will then start and download the created CSV file to your computer, in to the location specified in your browser.

The normal location for downloading, in the Chrome browser, is your Downloads folder.

Vehicle list

The vehicle list table shows details of each vehicle registered on the portal

The table shows the following columns:

  • The vehicle type, by use of an icon
  • The registration number
  • The UUID(s) of devices fitted to the the vehicle
  • The fleet the vehicle is assigned to
  • The site the vehicle is assigned to
  • The last online date and time
  • The status of the vehicle, on the portal
  • The On-Hire status (along with a flag on the left hand edge of the row)
  • Device status
    • Green: Device is online
    • Red: Device is offline
    • Muted device icon
    • Count of unacknowledged issues
  • Actions
    • Edit Vehicle
    • Remove Vehicle
    • View vehicle details

Actions

Edit

Clicking on the PENCIL icon will open the vehicle edit modal, that allows for editing of the vehicle details.

Remove Vehicle

Clicking on the TRASH icon will display the confirmation modal.

View vehicle details

Clicking on the VIEW icon will display the vehicle details page.

Please refer to Vehicle Details for further information

Pagination
