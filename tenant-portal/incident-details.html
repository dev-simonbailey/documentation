Incident Details

The Incident details page is divided into 2 areas

Map and Overview

Map

The map shows the location of the current incident.

All the normal map functions are available.

Overview

The overview section gives details on the following:

  • Icon showing the machine type
  • Registration of the vehicle
  • Date of the incident
  • Time of the incident
  • Incident Severity - Red / Yellow
  • The site incident occurred at
  • Camera number of the device
  • An Incident Video Download Request button
  • An Incident Image View icon

Incident Video Download Request

  • To request a download a video of the incident, click on the DOWNLOAD REQUEST button

  • Select the request type

  • and the reason for the request

  • Making a selection will send a video download request to the administrator

Play Downloaded Video

Clicking on the Play button will play the downloaded video

View Incident Image

Clicking on the Image icon will display the incident image
